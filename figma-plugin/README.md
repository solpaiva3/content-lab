# Content Lab — Figma Plugin

Populates a Figma template frame with structured post content generated by the Content Lab web app.

## How it works

1. Approve a post in the web app — a **Post ID** appears on the card
2. Copy the Post ID
3. In Figma, run the plugin, paste the ID, and click **Populate template**
4. The plugin fetches the content and fills in the text layers

## Installing locally in Figma

1. Open Figma Desktop (the plugin cannot be loaded from the browser)
2. Go to **Plugins → Development → Import plugin from manifest…**
3. Navigate to this `figma-plugin/` folder and select `manifest.json`
4. The plugin will appear under **Plugins → Development → Content Lab**

## Required Figma layer names

Your template frame must contain text layers with **exactly** these names (they can be nested anywhere inside the frame):

| Layer name        | Content          |
|-------------------|------------------|
| `post/title`      | Post title       |
| `post/subtitle`   | Subtitle / hook  |
| `post/body`       | Instagram caption|
| `post/logo-usage` | Logo instructions|

To rename a layer: double-click its name in the Layers panel.

## Usage

1. Select the template frame in Figma
2. Run **Plugins → Development → Content Lab**
3. Set the **Web App URL** (default: `http://localhost:3000`)
4. Paste the **Post ID** copied from the web app
5. Click **Populate template**

## Production deployment

Once the web app is deployed, update the default **Web App URL** in `ui.html` to your production URL.
